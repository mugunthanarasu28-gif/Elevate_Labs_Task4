# Binary Classification using Logistic Regression

## Overview
This project builds a binary classification model using Logistic Regression to predict class labels from the dataset. It includes data preprocessing, model training, evaluation, and performance analysis.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Tasks Performed
- Imported and preprocessed the dataset
- Split the data into training and testing sets
- Standardized numerical features
- Trained a Logistic Regression model
- Evaluated the model using Confusion Matrix, Precision, Recall, and ROC-AUC Score
- Plotted the ROC Curve
- Tuned the classification threshold
- Explained and visualized the Sigmoid Function

## Outcome
The model successfully performs binary classification and demonstrates how Logistic Regression can be used to make predictions and evaluate classification performance using standard metrics.
